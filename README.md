# 🧸 ToyWorld — Frontend React (Phía Người Xem)

Website bán đồ chơi trẻ em, xây dựng bằng **React + Vite**, thiết kế theo mockup tài liệu nhóm 2.

---

## 🚀 Cài đặt & Chạy

```bash
# 1. Di chuyển vào thư mục
cd ToyShop_FE

# 2. Cài dependencies
npm install

# 3. Chạy development server
npm run dev
# → Mở http://localhost:5173
```
